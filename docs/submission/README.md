# Submission package

This directory contains the Neural Networks submission-side files. The journal
uses single-anonymized review. Items in square brackets require confirmation
by the corresponding author.

Before submission:

1. Confirm the locked Neural Networks target and select “Learning Systems.”
2. Replace every bracketed placeholder in `title_page.tex` and
   `cover_letter_draft.md`.
3. Confirm funding, competing interests, author order, and CRediT roles with
   every author; do not infer these from repository commits.
4. Insert the public archival repository/DOI in `declarations.tex`.
5. Submit `supplementary_ablation.tex`, raw result archive, environment lock,
   and checksum manifest as supplementary/repository materials.
6. Upload `highlights.txt`; each of its five lines is within the journal's
   85-character limit.
